Functions

A basic function looks like
function nameOfFunction (){what to do}
we need to call the function where we want in our code like
nameOfFunction()
From ES6 we can use shorten version of functions like ()=>{}

function till(x, y){ return x * y}
till(4,5); will return 20

We can add a function to a variable
const till = (x, y)=>{ x * y} still calling like : till(4,5)
